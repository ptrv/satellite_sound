Creation / Class Methods


*new (satnumber, synthname, server, visualize, remoteport, remoteserver)

Short prose description of method.

satnumber - Total number of satellites.

synthname - Synthdef name to use.

server - Server. Default value is nil.

visualize - Set switch for remote visualization. Default is true.

remoteport - Remote visualization port. Default value is 12000.

remoteserver - Remote visualization server. Default value is "127.0.0.1".

// inline example

g = SatelliteSound.new(20);


Functions


start

Start synths.

// inline example

g = SatelliteSound.start;



stop

Stop synths.

// inline example

g = SatelliteSound.stop;



calculateNumberSequence (num1, num2, num3)

Returns pattern sequence..

num1 - Local latitude value.

num2 - Local longitude value.

num3 - Local eleation value.

// inline example

g = SatelliteSound(20);

a = g.calculateNumberSequence(52.090434, 13.44996, 1.3444);

a.postln; // explanation



changePatterns (paramName, array)

Changes synth pattern.

paramName - The control name.

array - Array with pattern values.

// inline example

g = SatelliteSound(0;

g.changePatterns(\dseq, [1,0,2,4,0,0,1,1]); // explanation




numActiveSatellites

returns number of active satellites.

// inline example

g = SatelliteSound.numActiveSatellites;
